# 项目代理规则

- 本项目中凡涉及本地模型、ASR、LLM 推理、音视频处理、转写、向量计算、批处理或其他可加速计算任务，只要运行环境支持 GPU，就必须默认启用 GPU 加速，以获得最大性能。
- 所有 GPU 加速能力必须同时提供 CPU 兼容 fallback；当 GPU 不可用、驱动缺失、CUDA/DirectML/Vulkan/OpenVINO 等运行时不可用或执行失败时，应自动降级到 CPU，并给出明确状态或错误记录。
- 配置项、启动脚本、下载脚本和运行时适配器应优先选择 GPU 版本依赖或二进制包；只有在明确检测不到 GPU 或用户显式选择 CPU 时，才使用 CPU-only 方案。
- 任何新增本地模型或第三方推理引擎时，必须把模型、缓存、日志和运行数据放在项目目录、用户指定的非 C 盘目录，或 `D:\Antigravity` / `D:\SoftInstall` 下，禁止默认写入 C 盘用户目录或系统盘。
